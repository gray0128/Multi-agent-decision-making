# Codex Agent 信息收集

本文件记录当前 Codex 运行环境公开给子 Agent 的模型与推理等级。该清单属于当前运行环境快照，不代表所有 Codex 账号或版本均可用。

## 1. 可用模型

| 模型 ID | 定位 | 默认 effort | Context | Max output |
|---|---|---|---:|---:|
| `gpt-5.6-sol` | 高能力 agentic coding | `low` | 1,050,000 | 128,000 |
| `gpt-5.6-terra` | 日常编码的均衡档 | `medium` | 1,050,000 | 128,000 |

窗口数据与 Models.dev 的 `openai/gpt-5.6-sol`、`openai/gpt-5.6-terra` 规范页一致；实际可用模型仍以当前 Codex 运行环境暴露的列表为准。

## 2. 推理等级

当前两个模型均支持：

`low`、`medium`、`high`、`xhigh`、`max`、`ultra`

推理等级控制单次任务投入，不改变模型声明的上下文窗口。子 Agent 未显式覆盖时继承父 Agent 的模型与 effort。
